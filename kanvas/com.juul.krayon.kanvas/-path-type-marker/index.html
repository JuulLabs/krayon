PathTypeMarker

interface PathTypeMarker<P : Any>

Used to bridge the gap between a Path and the platform type required by a Kanvas.

Ultimately, Path uses instances of this class as a Map key, so instances must correctly implement hashCode. This is automatic when implementations take the form of an object instead of a class.
